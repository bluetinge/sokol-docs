@page guide_headers Core Header Guides

Purposeful jump points for the main sokol headers.

- [sokol_app.h Guide](guide_app.html): window creation, callbacks, events, and platform integration.
- [sokol_gfx.h Guide](guide_gfx.html): renderer setup, resource creation, passes, and draw or compute entry points.
- [sokol_audio.h Guide](guide_audio.html): audio setup, push-model streaming, and runtime stream properties.
- [sokol_fetch.h Guide](guide_fetch.html): asynchronous file loading, request control, and buffer handoff.
- [sokol_args.h Guide](guide_args.html): command-line or query-string parsing helpers.
- [sokol_time.h Guide](guide_time.html): high-resolution timing and frame-pacing helpers.
- [sokol_glue.h and sokol_log.h Guide](guide_glue_log.html): convenience glue for module integration and default logging.
